# Olaniyi Lawal – Football Player Portfolio

A professional football portfolio website showcasing Olaniyi Lawal, a calm and confident central defender known for composure, strength, and excellent 1v1 defending.

## Features
- Player profile and bio
- Highlight videos
- Image gallery
- Contact section for trials and opportunities

## Tech Stack
- React / Vite
- Tailwind CSS

## Usage
1. Clone the repository
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`

## Note
This project is built as a reusable template for football player portfolios. Content (name, images, videos, bio) can be easily replaced to create new player profiles.
